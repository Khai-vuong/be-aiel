====AI CONVERSATIONS API DOCUMENTATION

Base URL: /ai
Authentication: Required (JWT Token in Authorization header)

====1. GET /ai/conversations
====Description: Lay danh sach tat ca conversations cua user hien tai
Method: GET
Role: any (authenticated users)

Query Parameters:
  - limit (optional): number - So luong conversation can lay (default: 20)
  - offset (optional): number - Vi tri bat dau (default: 0)

GET /ai/conversations?limit=20&offset=0

====3. PUT /ai/conversations/:id
====Description: Cap nhat thong tin conversation (title hoac status)
Method: PUT
Role: any (authenticated users, chi cap nhat duoc conversation cua minh)
URL Parameters:
  - id (required): string - Conversation ID
Content-Type: application/json

Request Body:
{
  "title": "Updated Title",
  "status": "archived"
}

Optional Fields:
  - title: string - Tieu de moi
  - status: 'active' | 'archived' - Thay doi trang thai

Example Request 1 (Update title):
PUT /ai/conversations/acid_clx123abc
{
  "title": "TypeScript Learning Guide"
}

Example Request 2 (Archive conversation):
PUT /ai/conversations/acid_clx123abc
{
  "status": "archived"
}

Example Output:
{
  "acid": "acid_clx123abc",
  "title": "TypeScript Learning Guide",
  "status": "active",
  "user_id": "uid_user123",
  "last_message_at": "2026-03-10T14:30:00Z",
  "metadata_json": null,
  "created_at": "2026-03-10T10:00:00Z",
  "updated_at": "2026-03-10T15:00:00Z"
}

Notes:
  - Co the cap nhat title hoac status rieng le hoac ca hai
  - Archive conversation de an khoi danh sach chinh (soft delete)
  - User chi cap nhat duoc conversation cua minh


====4. DELETE /ai/conversations/:id
====Description: Xoa conversation vinh vien (hard delete)
Method: DELETE
Role: any (authenticated users, chi xoa duoc conversation cua minh)
URL Parameters:
  - id (required): string - Conversation ID

Example Request:
DELETE /ai/conversations/acid_clx123abc

Example Output:
{
  "message": "Conversation deleted successfully"
}

Notes:
  - Day la hard delete - conversation va tat ca messages se bi xoa vinh vien
  - Khong the khoi phuc sau khi xoa
  - Nen dung archive (PUT status='archived') thay vi delete
  - User chi xoa duoc conversation cua minh (403 neu khong phai)
  - Cascade delete: Xoa conversation se tu dong xoa tat ca messages va summaries


====ERROR RESPONSES
====400 Bad Request
- Du lieu khong hop le
- Thieu required fields

Example:
{
  "statusCode": 400,
  "message": "Validation failed",
  "error": "Bad Request"
}


====401 Unauthorized
- Chua dang nhap
- Token khong hop le hoac het han

Example:
{
  "statusCode": 401,
  "message": "Unauthorized",
  "error": "Unauthorized"
}


====403 Forbidden
- Truy cap conversation cua user khac

Example:
{
  "statusCode": 403,
  "message": "You do not have access to this conversation",
  "error": "Forbidden"
}


====404 Not Found
- Khong tim thay conversation voi ID da cho

Example:
{
  "statusCode": 404,
  "message": "Conversation acid_xyz not found",
  "error": "Not Found"
}


====NOTES
====General Information

0. JWT token phai duoc gui trong Authorization header
   Format: Authorization: Bearer <token>

1. Conversation Status:
   - active: Conversation dang hoat dong
   - archived: Conversation da duoc archive (an di nhung van ton tai)

2. Message Role:
   - user: Tin nhan tu nguoi dung
   - assistant: Tin nhan tu AI
   - system: Tin nhan he thong
   - tool: Tin nhan tu tool/function call

3. Pagination:
   - Conversations: Su dung limit va offset
   - Messages: Su dung limit va beforeMessageId (load them message cu hon)

4. Auto-generated Title:
   - Khi tao conversation moi, title se tu dong tao tu 50 ky tu dau cua message dau tien
   - Co the cap nhat title bat ky luc nao

5. Token Tracking:
   - promptTokens, completionTokens, totalTokens chi co trong assistant messages
   - Dung de theo doi chi phi su dung AI

6. Metadata:
   - Co the luu them thong tin tuy chinh trong metadata field
   - Metadata la JSON object, co the chua bat ky du lieu gi

7. Message Threading:
   - parentMessageId cho phep tao thread message (reply to specific message)
   - Hien tai chua duoc su dung nhung da san sang cho tuong lai

8. Conversation Summary:
   - Voi conversation dai, he thong se tu dong tao summary
   - Summary giup toi uu context window khi gui den AI
   - API de lay summary: conversation.service.getConversationSummary()

9. Ownership:
   - Tat ca operations deu check ownership
   - User chi thao tac duoc voi conversation cua minh
   - Cascade delete: Xoa conversation se xoa tat ca messages lien quan

10. Best Practices:
    - Nen dung archive thay vi delete de giu history
    - Load messages theo batch (limit=50) de tranh qua tai
    - Su dung beforeMessageId de implement infinite scroll
    - Check totalMessages va hasMore de biet con message khong
